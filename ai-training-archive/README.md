# 📁 AI Training Archive

תיקייה זו מכילה את היסטוריית הלוגים ותיקוני ה-AI לפי אצוות.

## 🔄 תהליך העבודה

1. **איסוף** — השותף מכניס מוצרים ומוסיף הערות דרך הממשק
2. **ייצוא** — מהאדמין לוחצים "ייצוא CSV" → שולחים ל-AI לניתוח
3. **טיפול** — AI מנתח את השגיאות ומתקן את `listing-ai.ts`
4. **תיעוד** — AI מוסיף קובץ `batch-YYYYMMDD.md` לתיקייה זו עם סיכום השינויים
5. **ניקוי** — לוחצים "מחק הכל" באדמין להתחלה חדשה

## 📂 מבנה קבצים

```
ai-training-archive/
├── README.md              ← הקובץ הזה
├── batch-20260221.md      ← סיכום אצווה ראשונה (לדוגמה)
├── batch-20260221.csv     ← קובץ הנתונים המקורי
└── ...
```

## 📝 תבנית קובץ סיכום

כל קובץ `batch-YYYYMMDD.md` יכיל:
- תאריך הטיפול
- כמות הלוגים שנותחו
- שגיאות שזוהו (לפי קטגוריה)
- שינויים שבוצעו ב-`listing-ai.ts`
- דפוסים חדשים שנלמדו
