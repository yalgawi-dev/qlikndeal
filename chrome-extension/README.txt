# הוראות התקנה לתוסף Qlikndeal

מכיוון שהתוסף עדיין בפיתוח, מתקינים אותו ידנית:

1. פתח את דפדפן **Chrome**.
2. בשורת הכתובת, הקלד: `chrome://extensions` ולחץ Enter.
3. בצד שמאל למעלה (או ימין בעברית), הפעל את המתג **Developer mode** (מצב מפתח).
4. יופיעו כפתורים חדשים. לחץ על **Load unpacked** (טען ללא חבילה).
5. בחלון שנפתח, נווט לתיקייה של הפרויקט:
   `...\Qlikndeal\chrome-extension`
   ובחר בה.

זהו! התוסף יופיע ברשימה. כעת כנס לפייסבוק ותראה שמסגרת ירוקה מופיעה סביב פוסטים (סימן שהבוט מזהה אותם).
